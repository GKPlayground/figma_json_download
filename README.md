# Figma json download

## Start

- edit config.json

``` json
{
    "Token": "",
    "FileId": "",
    "NodeId": ""
}
```

- `go run main.go`

## About Token

<https://www.figma.com/developers/api#auth-dev-token>

## About FileId

Project URL
eg: <https://www.figma.com/file/WO5q4EGgdXC4rUGLnaqhok/Untitled?node-id=0%3A1>
